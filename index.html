To use this bookmarklet, simply drag one of the following to your bookmark bar:

  • HEAD version of rule_of_thirds.bookmarklet.js: RuleOfThirds
  • December 18, 2012 release of rule_of_thirds.bookmarklet.js: RuleOfThirds

The former will be kept up to date (new features, etc) while using the snapshot version is more stable (it won't stop working if I push a bad commit) and more secure (say if my github account gets compromised and malicious JS is pushed, it will not be injected into websites by bookmarklet on).
