To use this bookmarklet, simply drag one of the following to your bookmark bar:

  • December 18, 2012 release of rule-of-thirds.bookmarklet.js: RuleOfThirds
  • HEAD version of rule-of-thirds.bookmarklet.js: RuleOfThirds

The HEAD version is only provided for development purposes. The tagged version is more stable and more secure.That is, if broken or malicious JS code is pushed to HEAD, it will not be injected into your websites by this bookmarklet.
